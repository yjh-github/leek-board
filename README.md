# 基金看板工具

一个帮助您追踪基金每日估值和收益的工具。

## 技术栈

- **后端**: Node.js + Express + TypeScript + TypeORM + SQLite
- **前端**: React + TypeScript + Vite + Tailwind CSS

## 快速开始

### 一键启动（推荐）

双击运行 `start.sh` 脚本，或在终端执行：

```bash
./start.sh
```

脚本会自动：
1. 检查并安装依赖
2. 在新终端窗口启动后端服务
3. 在新终端窗口启动前端服务

### 手动启动

#### 1. 启动后端服务

```bash
cd server
npm run dev
```

后端服务将在 http://localhost:3001 启动。

#### 2. 启动前端开发服务器

打开新的终端窗口：

```bash
cd client
npm run dev
```

前端应用将在 http://localhost:5173 启动。

## 功能说明

### 基金管理
- **添加基金**：输入基金代码、持仓成本、持有份额，名称自动获取
- **编辑基金**：修改持仓成本和份额
- **删除基金**：移除不需要的基金

### 数据展示
- 持仓成本显示 5 位小数
- 最新净值显示 4 位小数
- 收益、收益率实时计算
- 日涨跌幅显示

### 排序功能
点击表头可按以下字段排序：
- 日涨跌
- 当前价值
- 收益
- 收益率

### 自动刷新
- 勾选"自动刷新"启用定时刷新
- 可选间隔：30秒、1分钟、5分钟、10分钟

### 数据导入/导出
- **导出**：点击"导出"按钮，下载 JSON 格式备份文件
- **导入**：点击"导入"按钮，选择之前导出的 JSON 文件恢复数据

## 常见问题

### 后端启动失败
- 确认已在 server 目录下安装了依赖：`npm install`
- 检查端口 3001 是否被占用

### 前端无法访问后端
- 确认后端已在 3001 端口启动
- 检查浏览器控制台的错误信息

### 基金数据获取失败
- 检查网络连接
- 确认基金代码正确（天天基金网的基金代码）
